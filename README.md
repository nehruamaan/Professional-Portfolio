# Amaan Nehru — Professional Portfolio

Personal portfolio website built as a single-page HTML/CSS/JS application. No frameworks, no build tools — just clean, self-contained files.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main portfolio — hero, featured work, experience, skills, education, contact |
| `case-study-bi.html` | Business Insider case study |
| `case-study-efl.html` | EFL case study |

## Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Barlow, Barlow Condensed, Inter)
- Formspree for contact form
- No dependencies, no build step

## Sections (index.html)

- **Hero** — tagline, focus areas (AI Strategy · Product Thinking · GTM Execution · Venture & Consulting)
- **Featured Work** — selected case studies
- **How I Think** — mental operating system / frameworks
- **Experience** — career history
- **Capabilities** — skills breakdown
- **Education** — academic background
- **Contact** — message form

## Running locally

Open any `.html` file directly in a browser. No server required.
